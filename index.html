hyperctl Downloader

Install the latest hyperctl CLI with a single command. 

curl -fsSL https://dl.openhack.org/hyperctl_install.sh | sh

The installer downloads the newest hyperctl release from GitHub and places it in /usr/local/bin by default.


hyper-cli Downloader

Install the latest hyper-cli with a single command.
