hyperctl Installer

Install the latest hyperctl using the installer script. 

curl -fsSL https://dl.openhack.org/hyperctl_install.sh | sh

The installer downloads the newest release and places it in /usr/local/bin by default.

Binaries are pulled from our GitHub releases: OpenLabsRo/openhack-hypervisor.
