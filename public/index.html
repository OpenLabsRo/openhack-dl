OpenHack CLI Downloads

Install the latest hyperctl CLI with a single command.

One-liner install

Run the following command on a Linux amd64 host:

curl -fsSL https://dl.openhack.org/hyperctl_install.sh | sh

The installer downloads the newest hyperctl release from GitHub and places it in /usr/local/bin by default.

Advanced usage

Override defaults by exporting environment variables before running the installer:

INSTALL_DIR=$HOME/.local/bin curl -fsSL https://dl.openhack.org/hyperctl_install.sh | sh

Available variables: OWNER, REPO, ASSET_NAME, BINARY_NAME, INSTALL_DIR.

Release source

Binaries are pulled from the latest GitHub release at openlabs-hq/hyperctl.
